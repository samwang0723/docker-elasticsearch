# docker-elasticsearch
Elasticsearch within docker, Elasticsearch version `v6.5.4`

## Command

### Build image
```sh
docker-compose build
```

### Run image
```sh
docker-compose up
```

### Shutdown image
```sh
docker-compose down
```

## Default build-in analyzer

ik-smart, kuromoji
